# mtg-volodex

CLI companion app to browse and discover new creature types for your [Volo, Guide to Monsters](https://scryfall.com/card/afr/238/volo-guide-to-monsters) Commander deck.

## Getting Started

```
    $ ./install.sh
    $ ./volodex.sh download
    $ ./volodex.sh update
    $ ./volodex.sh run
```
